Explore Computing with Short Courses
For those interested in an introduction to computer science topics without a long-term commitment, our department offers affordable, flexible short courses. Taught by expert faculty in accelerated formats, these courses allow hands-on exploration of concepts and skills.

 Cybersecurity Fundamentals
A week-long online course covers basic security principles, common threats, and protective measures.
Ethical Hacking
A hands-on course teaching lawful hacking techniques like penetration testing and vulnerability assessment. Conduct security audits and risk analyses.
Mobile Application Development
An accelerated course to develop iOS or Android apps, introducing Swift/Java, UI/UX design, and publishing to app stores. Introduction to Web Programming
A project-oriented short course to design, code and deploy basic websites using HTML/CSS, JavaScript and a web framework.

Whether starting a new career or enhancing current job skills, short courses provide an accessible introduction to cutting-edge computer science disciplines. Earn a credential and see if more in-depth education is right for you.
